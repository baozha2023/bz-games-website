你的游戏，你的规则

BZ-Games 是一个无服务器的本地游戏平台。导入、管理、联机 — 一切都存储在你自己的电脑上，无需云端账号。

下载 Windows 版 GitHub

为什么选择 BZ-Games

专为 Windows 打造的本地游戏管理体验

📂

开放式游戏库

支持导入任意符合规范的本地游戏，自动识别版本、封面与元数据，拖拽即玩。

🔌

无服务器架构

所有数据存储在你的电脑上。游戏记录、成就、BZ 币 — 完全私有，无需注册云端账户。

🎮

统一联机大厅

内置房间系统：创建 / 加入 / 准备 / 开始。游戏开发者无需编写网络代码。

🌐

灵活内网穿透

通过标准 TCP 端口暴露房间，支持 SakuraFrp 等任意内网穿透工具，不锁定特定服务。

🏪

游戏市场

浏览并一键下载社区游戏。自动校验 SHA-256、解压、导入 — 全程无需手动操作。

🏆

成就与经济

游戏内解锁成就、累计 BZ 币、每日签到。平台统一追踪你的游戏成就与统计。
